# 项目初始化

## 环境准备

```bash

# node 16+

# 全局安装pnpm
npm install -g pnpm

```

## 创建项目

```bash

# 创建项目
pnpm create vite

# 选择 vue 项目
? Project name Vue3-TS-poject01
? Select a framework: vue
? Select a variant: TypeScript

#进入项目目录
cd Vue3-TS-poject01 # Vue3-TS-poject01是创建的项目名称

# 安装依赖
pnpm install

#运行项目
pnpm run dev

```
